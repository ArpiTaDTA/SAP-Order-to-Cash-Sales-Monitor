| Module | Topic | Where It Appears In Project |
|---|---|---|
| Module 01 | ERP and Digital Enterprise | Project problem statement and reporting use case |

| Module 05 | Organizational Structure | Company Code, Plant, Sales Org, Distribution Channel, Division |

| Module 11 | Reporting / Analytics / CDS | KPI cards, operational reporting, analytics discussion |

## Why This Project Fits The Modules

- It uses a familiar SAP business scenario: sales orders in OTC flow.
- It demonstrates both functional and technical understanding.
- It is easy to explain during project evaluation because every screen connects back to a module topic.
